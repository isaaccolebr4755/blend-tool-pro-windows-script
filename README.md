## What it does

Blend Tool Pro is a JavaScript utility built for Affinity V3.2 that helps you construct controlled blends across vector shapes, grouped objects, and paths. It is geared toward generating intermediate steps, shaping the transition between source elements, and deciding how the final blended result is assembled in the document.

The script is centered on hands-on blend refinement. It includes options for guide paths, easing behavior, twist handling, and object matching, giving users more control over transition layout while staying within Affinity's scripting workflow.

## Core capabilities

- Support for blending shapes and groups
- Path-driven blending with an editable spine
- Twist and easing settings for shaping the transition
- Fill and stroke interpolation controls
- Live non-destructive preview while tuning settings
- Anchor matching for more consistent object pairing
- Re-stepping tools for iterative refinement
- Stacking order control for generated blend output
